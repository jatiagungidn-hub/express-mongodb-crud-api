# Express MongoDB CRUD API

A simple RESTful CRUD API built with Express.js and MongoDB using Mongoose.

## Features

- Create user
- Get all users
- Get user by ID
- Update user
- Delete user
- Filter users by name, age, and role

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
MONGO_URI=your_connection_string
PORT=3000
```

## Run

```bash
npm start
```
